How to Use This Dashboard

1. Start with OverviewUse the four priority cards and status lists to identify delivery pressure quickly.
2. Filter the PortfolioYear, quarter, cluster, LGA, status, funding source, and search apply across relevant views.
3. Drill Into ProjectsClick any project row or card to view location, dates, progress, financials, outputs, outcomes, contractor details, images, and milestone comments.
4. Read Status LogicQuarter analysis uses each project start and due date. Projects behind expected progress are marked At Risk.
5. Interpret ImpactCitizen impact and SDG alignment sit under Outcome/Impact Performance, away from the overview KPI cards.
